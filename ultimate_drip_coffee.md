# PART 1: 물 (Water Specialist)

## 1.1 최적의 물 품질

### SCA 기준 요약

| 항목 | 목표치 | 허용 범위 |
|------|--------|-----------|
| TDS | 150 ppm | 75~250 ppm |
| 총경도 | 68 ppm | 50~175 ppm |
| 알칼리도 | 40 ppm | 40~70 ppm |
| pH | 7.0 | 6.5~7.5 |
| 나트륨 | 10 mg/L | 10 mg/L 이하 |
| 염소 | 0 mg/L | 0 mg/L |

### 이상적인 미네랄 구성

- **칼슘**: 크리미한 바디감과 풍부한 마우스필 부여. 권장 50~175 ppm (CaCO3 기준)
- **마그네슘**: 과일향, 산미, 달콤한 뉘앙스 추출에 탁월. 칼슘보다 커피 풍미 화합물과의 결합력이 더 강함
- **칼슘:마그네슘 비율**: 일반적으로 60:40 권장. 라이트 로스트는 마그네슘 비율을 높이고, 다크 로스트는 칼슘 비율을 높이는 것이 유리

### 완벽한 물 체크리스트

| 항목 | 최적 수치 |
|------|-----------|
| TDS | 100~150 ppm |
| 총경도 | 50~85 ppm |
| 칼슘 | 30~50 ppm |
| 마그네슘 | 20~40 ppm |
| 알칼리도 | 40~70 ppm |
| pH | 6.5~7.5 (목표 7.0) |
| 염소 | 0 ppm |

## 1.2 물 온도의 과학

### 왜 92~96°C가 최적인가

| 온도 범위 | 주요 추출 성분 | 맛 특성 |
|-----------|---------------|---------|
| 85°C 이하 | 산(acids), 과일향 일부 | 신맛 지배적, 단맛과 바디 부족 |
| 85~90°C | 산 + 일부 당류 | 밝은 산미는 있으나 바디감 부족 |
| **92~96°C** | **산 + 당류 + 오일 + 풍미 화합물** | **산미, 단맛, 바디의 최적 균형** |
| 96°C 초과 | 쓴맛 화합물, 탄닌, 과도한 오일 | 쓰고 거칠며 섬세한 풍미 파괴 |

**추출 순서**: 산성 물질(과일산) → 당류(단맛) → 오일(바디) → 쓴맛 물질(탄닌) 순으로 추출되므로, 92~96°C는 바람직한 성분은 충분히, 불쾌한 성분은 최소화하는 "골든 존"

## 1.3 물 종류별 비교

| 종류 | TDS | 장점 | 단점 | 평가 |
|------|-----|------|------|------|
| 수돗물 | 50~500+ ppm | 무료, 쉬운 접근 | 염소, 지역 편차 | 정수 필수 |
| **정수물** | 조절 가능 | **염소 제거 + 미네랄 보존** | 필터 교체 필요 | **대부분에게 최고** |
| 생수 | 50~157 mg/L | 일관된 수질 | 비용, 환경 부담 | 좋은 대안 |
| 증류수 | 0~5 ppm | 미네랄 첨가 베이스 | 단독 사용 불가 | 첨가제와 조합 시 최상 |

## 1.4 프로 바리스타의 물 레시피

### Third Wave Water 프로파일

| 프로파일 | TDS | 알칼리도 | 적합한 용도 |
|----------|-----|----------|-------------|
| Classic Light Roast | ~150 mg/L | ~40 mg/L | 푸어오버, 프렌치프레스 |
| Medium Roast | ~150 mg/L | ~80 mg/L | 미디엄 로스트 전용 |
| Dark Roast | ~180 mg/L | ~135 mg/L | 다크 로스트 전용 |

### DIY 미네랄 워터 (Barista Hustle 방식)

**농축액 A (마그네슘)**: 엡솜 솔트 2.45g + 증류수 1L
**농축액 B (버퍼)**: 베이킹 소다 1.68g + 증류수 1L

브루잉 물 1L = 농축액 A 50~75mL + 농축액 B 25~50mL + 나머지 증류수 → **갤런당 약 1센트**

## 1.5 온도 유지 팁

1. **장비 예열 필수**: 드리퍼, 서버, 컵 모두 뜨거운 물로 린싱
2. **플라스틱 드리퍼가 보온 최강**: 열방출 속도가 다른 재질의 1/20 → 많은 대회 바리스타가 선호
3. **온도 조절 전기 케틀 사용**: Fellow Stagg EKG 등으로 목표 온도 자동 유지
4. **주수 높이를 낮게**: 물이 공기 중에서 냉각되는 시간 최소화
5. **겨울철 보정**: 평소보다 1~2°C 높은 온도로 시작

# PART 2: 원두 (Bean Master)

## 2.1 드립에 최적의 원두 산지

### 에티오피아 ★★★★★
- **대표**: 예가체프(Yirgacheffe), 시다마(Sidama), 구지(Guji)
- **풍미**: 꽃향기(자스민, 베르가못), 과일 산미(블루베리, 복숭아), 차(tea) 같은 목넘김
- **드립 적합도**: 최상 - 가장 복합적이고 화려한 향미 구현

### 콜롬비아 ★★★★★
- **대표**: 우일라(Huila), 나리뇨(Narino)
- **풍미**: 캐러멜, 견과류 단맛, 밝은 산미, 부드러운 바디
- **드립 적합도**: 최상 - 입문자부터 전문가까지 모두 적합

### 케냐 ★★★★☆
- **대표**: 케냐 AA, 케냐 AB
- **풍미**: 강렬한 산미(자몽, 블랙커런트), 와인 같은 풍부함
- **드립 적합도**: 산미를 즐기는 이에게 최고. 호불호 가능

### 과테말라 ★★★★☆
- **대표**: 안티구아(Antigua), 우에우에테낭고
- **풍미**: 다크 초콜릿, 스모키, 딸기, 묵직한 바디
- **드립 적합도**: 초콜릿 계열 선호자에게 추천

### 기타 주목 산지
- **파나마 게이샤**: 세계에서 가장 비싼 커피. 꽃향, 열대과일이 압도적
- **코스타리카 타라주**: 밝은 산미, 깔끔한 바디, 꿀 같은 단맛
- **브라질 세하도**: 견과류, 초콜릿, 낮은 산미로 부드러운 드립에 적합

## 2.2 로스팅 레벨

### 드립에 가장 추천: 미디엄 로스팅 ★★★★★

| 로스팅 | 온도 | 맛 특성 | 드립 적합도 |
|--------|------|---------|------------|
| 라이트 | 180~205°C | 생두 고유 향미, 밝은 산미, 가벼운 바디 | 스페셜티 테루아 체험에 최적 |
| **미디엄** | **210~220°C** | **산미+단맛+바디 조화, 캐러멜, 초콜릿** | **드립의 표준** |
| 다크 | 225~240°C | 진하고 쓴 맛, 스모키, 산지 특성 소실 | 진한 맛 선호 시 |

> **하이~시티 로스팅 단계**에서 산미와 단맛, 바디감이 가장 조화롭게 표현됨

# PART 3: 추출 테크닉 (Technique Master)

## 3.1 프로 바리스타 주수 기법

### 4:6 메서드 (테츠 카스야, 2016 WBrC 챔피언)

> 경험보다 **공식**으로 맛을 제어하는 혁신적 방법

- **비율**: 커피 20g : 물 300ml (1:15)
- **공식**: 커피 무게 x 3 = 1회 주수량 → 60g x 5회
- **Phase 1 (1~2차, 40%)**: 산미/단맛 밸런스 조절
  - 1차 > 2차 → 산미 강화 (예: 70g + 50g)
  - 1차 < 2차 → 단맛 강화 (예: 50g + 70g)
- **Phase 2 (3~5차, 60%)**: 농도(바디) 조절
  - 3회 분할 → 진한 바디
  - 2회 분할 → 가벼운 바디
- **핵심**: 매 주수 후 물이 완전히 빠질 때까지 대기

### 라오 스핀 (Scott Rao)

- **방법**: 주수 직후 드리퍼를 부드럽게 2~3회 원형으로 스월
- **목적**: 채널링 최소화, 균일 추출, 평탄한 커피 베드 형성
- **타이밍**: 블루밍 후 1회, 최종 주수 후 1회

### 일본식 점적법 (点滴法)

- **방법**: 중앙에만 물을 한 방울씩 떨어뜨리는 농축 추출 후 희석
- **맛**: 넬드립과 같은 단맛과 바디, 식어도 맛이 떨어지지 않음
- **난이도**: 수십 년간 수련이 필요한 장인 기법

### 기법 비교

| 기법 | 추출 시간 | 맛 프로파일 | 난이도 | 재현성 |
|------|----------|------------|--------|--------|
| 점적법 | 5~10분+ | 농축 단맛, 풍부한 바디 | 매우 높음 | 낮음 |
| **4:6 메서드** | **3:30~4:00** | **조절 가능** | **낮음** | **매우 높음** |
| 라오 스핀 | 2:30~3:30 | 깔끔, 균일 | 중간 | 높음 |
| 연속 주수 | 2:30~3:00 | 부드러운 바디 | 중간 | 중간 |

## 3.2 뜸 들이기(Blooming)의 과학

### CO2의 문제
물 접촉 시 CO2가 급격히 방출 → **소수성 장벽 형성** → 물의 침투 차단 → 불균일 추출

### 로스팅별 최적 뜸 시간

| 로스팅 | 최적 뜸 시간 | 이유 |
|--------|-------------|------|
| 라이트 | **45~60초** | 밀도 높아 CO2 방출 느림 |
| 미디엄 | **30~45초** | 균형 잡힌 방출 |
| 다크 | **20~30초** | 다공성 증가로 빠른 방출 |

- **물의 양**: 커피 무게의 2배 (20g → 40ml)
- **교반**: 스월링 권장 (스터링보다 재현성 우수, 추출률 최대 25% 향상)

## 3.3 물줄기 제어

| 요소 | 최적값 | 효과 |
|------|--------|------|
| 유량 | **3~4g/초** | 균일한 추출 밸런스 |
| 높이 | **3~5cm** | 적당한 교반 + 온도 유지 |
| 패턴 | **나선형(Spiral)** | 가장 균일한 추출 |

## 3.4 펄스 주수 vs 연속 주수

| 항목 | 펄스 주수 | 연속 주수 |
|------|----------|----------|
| 추출 수율 | 21~23% | 18~19% |
| 균일성 | 30% 더 균일 | 보통 |
| 맛 | 밝은 산미, 깨끗한 맛 | 부드럽고 라운드한 바디 |
| 적합 원두 | 워시드 에티오피아, 게이샤 | 브라질 내추럴, 수마트라 |

## 3.5 추출 수율 (Extraction Yield)

### SCA 골든컵 스탠다드
- **추출 수율**: 18~22%
- **TDS**: 1.15~1.35%

```
추출 수율(%) = (완성 커피 무게(g) x TDS(%)) / 원두 무게(g)
```

예: 원두 20g, 완성 커피 300g, TDS 1.35% → (300 x 0.0135) / 20 = **20.25%**

### 조절 변수

| 변수 | 수율 높이려면 | 수율 낮추려면 |
|------|-------------|-------------|
| 분쇄도 | 곱게 | 굵게 |
| 물 온도 | 높게 (최대 96°C) | 낮게 (최소 85°C) |
| 추출 시간 | 길게 | 짧게 |
| 교반 | 강화 | 최소화 |

> **핵심**: 한 번에 하나의 변수만 조절하여 실험

## 3.6 세계 대회 우승자 레시피

### 2025 WBrC 챔피언 - 조지 펑 (George Peng)

| 항목 | 상세 |
|------|------|
| 드리퍼 | Solo Dripper |
| 원두 | 파나마 Mount Totumas 게이샤 (94.5점) |
| 원두량 | 15g |
| 물 TDS | 40 PPM |
| 1차 주수 | 30g, 96°C |

```
[0:00] 린싱 & 예열 → 린싱물 버리기
       원두 15g 투입, 저울 영점

[0:00] 1차 주수: 45g (93°C)
       중심에서 바깥으로 나선형, 스월링 1회
       → 45초 대기 (뜸 들이기)

[0:45] 2차 주수: 50g (누적 95g)
       작은 원을 그리며 부어줌
       → 물이 빠질 때까지 대기

[1:15] 3차 주수: 50g (누적 145g)
       일정한 유량(3~4g/초)으로 나선형
       → 대기

[1:45] 4차 주수: 50g (누적 195g)
       → 대기

[2:15] 5차 주수: 45g (누적 240g)

       → 완전 배수 대기
```
